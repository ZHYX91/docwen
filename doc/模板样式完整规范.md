# 模板样式规范

本文档记录多语言 DOCX 模板的样式规范设计来源和技术实现说明。

> **注意**：标题样式和正文段落样式现已由代码自动注入，无需手动修改模板文件。
> 详细格式配置见 `src/docwen/i18n/locales/*.toml` 的 `[style_formats]` 节。

## 二、技术实现说明

### 样式自动注入机制

样式注入器（`injector.py`）会在 MD 转 DOCX 时自动检测并注入缺失的样式：

1. **标题样式 (heading 1-9)**
   - 由 `get_localized_heading_template()` 根据语言配置动态生成
   - 格式配置从 `[style_formats.heading_1]`、`[style_formats.heading_2]`、`[style_formats.heading_3_9]` 读取

2. **正文段落样式**
   - 由 `get_localized_body_paragraph_template()` 动态生成
   - 格式配置从 `[style_formats.body_paragraph]` 读取
   - 样式名从 `[styles].body_paragraph` 读取

3. **自定义样式（代码/引用/公式等）**
   - 由 `get_localized_style_configs()` 获取本地化名称
   - 样式名从 `[styles]` 节读取

### 模板文件要求

模板文件只需提供基础文档结构（页面设置、页眉页脚等），样式定义由代码自动补充。
如果模板中已存在某样式，代码会跳过注入，保留原有样式。

## 三、Word XML 字体属性参考

### 字体类型说明

- **东亚字体 (eastAsia)**：用于中日韩字符，按目标用户地区设置
- **西文字体 (ascii/hAnsi)**：统一使用 Times New Roman（正文）或 Arial（标题）
- **复杂脚本字体 (cs)**：用于阿拉伯语、希伯来语等

### XML 属性示例

```xml
<w:rFonts 
    w:ascii="Times New Roman"      <!-- ASCII字符（基本拉丁字母） -->
    w:hAnsi="Times New Roman"      <!-- 高位ANSI字符（扩展拉丁字母） -->
    w:eastAsia="宋体"              <!-- 东亚字符（中日韩） -->
    w:cs="Times New Roman"         <!-- 复杂脚本（阿拉伯语等） -->
/>
```

### 单位换算参考

| 单位 | 说明 | 换算 |
|------|------|------|
| 半点 (half-point) | `w:sz` 属性使用 | 16pt = 32 |
| 字符 (1/100) | `w:firstLineChars` 属性使用 | 2字符 = 200 |
| twip | `w:firstLine`、`w:before`、`w:after` 属性使用 | 1pt = 20twip, 1cm ≈ 567twip |
